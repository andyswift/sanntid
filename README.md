Sanntid.js
==
Sanntid.js is a node.js module for getting real-time data from the public Ruter API.

## Installation
Simply do

    npm install sanntid

## How to work it

    ./sanntid.js <locationID>

Or you can search for a location:

	./sanntid.js sofienberg
